Data_cleaning_and_classifiers is my final project in the course of data science by Leonid Serkin at School of Sciences (UNAM). I cleaned a dataset of 125,000 rows with integrity problems like missing values, outliers, inconsistent formatting, data type mismatch, class imbalance, etc using Pandas, Matplotlib, NumPy, and I performed a trial-and-error process in order to obtain the best outcomes that involved PCA, log normalization and dropping arbitrary rows (majority class). Moreover, I trained a RandomForestClassifier, MLP and a DecisionTreeClassifier to make a classification algorithm. There are comments throughout the document that explain every step in the code and can run in Jupyter.

Espectro_LIBSde_Moneda_de_10_pesos_mexicanos is a Jupyter Notebook designed to extract and analyze spectral data (wavelength and intensity) obtained via LIBS (Laser-Induced Breakdown Spectroscopy) in air on coins and copper targets. The code processes CSV files generated by Ocean Optics software, enabling it to read, extract, and perform statistical calculations—such as computing the average intensity across 20 files of the same material. Furthermore, it automatically generates intensity spectrum plots (Intensity vs. Wavelength) for each material analyzed.
