Data_cleaning_and_classifiers is my final project in the course of data science by Leonid Serkin at School of Sciences (UNAM). I cleaned a dataset of 125,000 rows with integrity problems like missing values, outliers, inconsistent formatting, data type mismatch, class imbalance, etc using Pandas, Matplotlib, NumPy, and I performed a trial-and-error process in order to obtain the best outcomes that involved PCA, log normalization and dropping arbitrary rows (majority class). Moreover, I trained a RandomForestClassifier, MLP and a DecisionTreeClassifier to make a classification algorithm. There are comments throughout the document that explain every step in the code and can run in Jupyter.

Espectro_LIBSde_Moneda_de_10_pesos_mexicanos is a Jupyter Notebook designed to extract and analyze spectral data (wavelength and intensity) obtained via LIBS (Laser-Induced Breakdown Spectroscopy) in air on coins and copper targets. The code processes CSV files generated by Ocean Optics software, enabling it to read, extract, and perform statistical calculations such as computing the average intensity across 20 files of the same material. Furthermore, it automatically generates intensity spectrum plots (Intensity vs. Wavelength) for each material analyzed. 
Note: This script is configured to run specifically with my raw CSV files stored in Google Drive. If you would like to test the code or reproduce the results, please contact me, and I will gladly share the access links.

The project "Prototipo de maquina recibidora e iluminadora de billetes" is an arduino script to emulate the behave of a counterfeit banknote detector machine, using a servomotor and LCD electronic screen. The algorithm consist in three steps: receive ticket, analize ticket (leds and screen turning on), veredic (True/False), return ticket (leds and screen turning off),and the new cycle can start. this project was planning to be counterfeit banknote detector machine with CNN (convoultional-neural-networks) but the final script is not complet now, for this reason it's not include in the repository. The code can run in a arduino simulator like thinkercad.


ServicioSocialEstabilidadAMP is a Python script designed to analyze data regarding the position, size, and levitation time of acoustically levitated drops. Utilizing Pandas, Matplotlib, and NumPy, the script reads raw CSV files, cleans the data, and plots size and position against time. These visualizations showcase the evolution of the droplet during the preconcentration process—where a laser heats the droplet to decrease its volume—effectively analyzing its stability.
Note: Similar to the "Espectro_LIBSde_Moneda_de_10_pesos_mexicanos" project, this script requires specific raw CSV files to run. If you would like to test the code, please feel free to contact me, and I will gladly provide the necessary data files.
