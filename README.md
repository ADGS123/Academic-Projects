## Physics Simulations & Data Analysis

### Simulación de Patrones de Difracción
* **Description:** A Python script that simulates and represents X-ray diffraction (XRD) patterns to identify the atomic structures of $\text{Cu}$, $\text{Cu}_2\text{O}$, and $\text{CuO}$.
* **Technical Details:** Implements physical calculations for structure and shape factors in solids—physico-mathematical tools used to determine XRD intensity via Miller indices. 
* **Outputs:** Generates diffraction pattern plots (Intensity vs. $2\theta$) and maps the reciprocal space to analyze the structural features of these materials.

### Espectro LIBS de Moneda de 10 Pesos Mexicanos
* **Description:** A Jupyter Notebook designed to extract and analyze spectral data (wavelength and intensity) obtained via LIBS (Laser-Induced Breakdown Spectroscopy) in air on coins and copper targets.
* **Technical Details:** Processes raw CSV files generated by Ocean Optics software to compute statistical calculations, such as the average intensity across 20 files of the same material, and automatically plots Intensity vs. Wavelength.
* **Note:** This script relies on raw data stored in Google Drive. If you would like to test the code or reproduce the results, please contact me and I will gladly grant access.

### Servicio Social: Estabilidad AMP
* **Description:** A Python script developed during my social service to analyze the position, size, and levitation time of acoustically levitated droplets.
* **Technical Details:** Uses **Pandas, NumPy, and Matplotlib** to clean raw CSV data and plot droplet size and position against time. It visualizes droplet evolution during preconcentration, where a laser heats the droplet to decrease its volume, effectively analyzing its stability.
* **Note:** This script requires specific raw CSV files. Feel free to contact me if you need the data files to run the code.

## Hardware & Embedded Systems

### Banknote Validator Prototype (Arduino)
* **Description:** An Arduino script that simulates the behavior of a counterfeit banknote detection machine using a servomotor and an LCD screen.
* **Workflow:** The algorithm follows a 4-step cycle: Receive banknote $\rightarrow$ Analyze banknote (LEDs and screen activation) $\rightarrow$ Verdict (True/False) $\rightarrow$ Return banknote (LEDs and screen off).
* **Future Work:** This project was originally intended to integrate a Convolutional Neural Network (CNN) for automated detection. The CNN model is currently under development and is not yet included in this repository.
* **Environment:** The code can be executed using Arduino simulators like **Tinkercad**.
